Calculadora em Python (CLI)

Este projeto consiste em uma calculadora desenvolvida em Python para execução via linha de comando (CLI).
O programa permite realizar operações matemáticas básicas de forma interativa, contando com validação de entradas, tratamento de erros e execução contínua até que o usuário decida encerrar.

- Funcionalidades

Soma, subtração, multiplicação e divisão
Validação dos números inseridos pelo usuário
Validação dos operadores permitidos
Tratamento de erros utilizando try/except
Execução em loop até o usuário optar por sair

- Tecnologias utilizadas

Python 3
Estruturas condicionais (if, elif, else)
Laços de repetição (while)
Operadores lógicos e de comparação
Tratamento de exceções

- Como executar o projeto?

* Certifique-se de ter o Python 3 instalado
{Clone este repositório
Execute o arquivo no terminal com o comando:
python calculator.py}

- Objetivo do projeto

Projeto desenvolvido com foco em aprendizado, visando o fortalecimento da lógica de programação, validação de dados e aplicação de boas práticas iniciais em Python.
