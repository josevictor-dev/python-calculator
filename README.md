# Calculadora em Python (CLI)

Este projeto consiste em uma calculadora desenvolvida em Python para execução via linha de comando (CLI).
O programa permite realizar operações matemáticas de forma interativa, com validação de entradas,
tratamento de erros e estruturação utilizando funções.

## Funcionalidades
- Soma (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/), com tratamento de divisão por zero
- Potenciação (^)
- Validação dos números inseridos pelo usuário
- Validação dos operadores permitidos
- Tratamento de erros utilizando `try/except`
- Execução contínua em loop

## Tecnologias utilizadas
- Python 3
- Funções
- Estruturas condicionais (`if`)
- Laços de repetição (`while`)
- Operadores lógicos e de comparação
- Tratamento de exceções

## Como executar o projeto
1. Certifique-se de ter o **Python 3** instalado
2. Clone este repositório
3. Execute o arquivo no terminal:
   ```bash
   python calculadora_final.py
